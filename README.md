# 📊 Data Table

A responsive and reusable React Data Table component built with modern web technologies. It provides an intuitive interface for displaying tabular data with a clean UI and scalable component structure.

## ✨ Features

- 📋 Responsive data table layout
- 🔍 Search functionality
- ↕️ Sortable columns
- 📄 Dynamic data rendering
- 🎨 Clean and modern UI
- ⚡ Built with reusable React components

## 🛠️ Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/sonianuj01/Data-Table.git
```

Navigate to the project:

```bash
cd Data-Table
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open your browser at:

```
http://localhost:5173
```
